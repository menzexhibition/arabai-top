Current status

Recorded now, redeemable later

Why some tasks cost more

A short answer is like a snack. A long document is like a full meal. An image is like a decorated cake. A video is many pictures moving together, so the kitchen works much harder.

When to use what

ARABAI should not replace official websites. It should help users pick the easiest route.

Use official sites

For full product access

Use the official website when you need the full app, team plan, or official subscription.

ARABAI Credits

For quick tasks

Use credits later when you want one simple place for common AI work without opening many accounts.

Use task platform

For done-for-you work

Use a separate task platform when you want another person to complete the AI task for you.

Important rule

ARABAI should stay helpful first. Credits should be recorded and explained clearly, and should not become a forced pop-up or a payment wall.

What stays free

ARABAI articles, tutorials, prompts, examples, and bilingual guides stay free. Credits are only for signed-in users who later want to run AI tools from one place or receive approved contribution rewards.
