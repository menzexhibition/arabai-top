# ARABAI Business Model

ARABAI is a free AI guide and practical tool entry point for ordinary users.

The site should not charge for tutorials, articles, prompts, templates, or bilingual content.

## What Stays Free

All guide content stays free:

- beginner explanations
- real task tutorials
- screenshots and examples
- prompt templates
- English content
- Arabic content
- Chinese owner-side drafts and planning

The main purpose of free content is trust:

```text
People learn AI here.
People know which tool to try.
People come back when they need help.
```

## Revenue Line 1: Official Tool Links

For official tools, ARABAI sends users to the official website.

Examples:

- ChatGPT
- Claude
- Gemini
- Gamma
- Canva
- Runway
- Kling
- Suno
- Udio

When available, ARABAI can use:

- affiliate links
- referral links
- partner links
- tracked outbound links

Rules:

- Always label official links clearly.
- Do not pretend ARABAI owns the official tool.
- Do not sell official subscriptions unless there is a formal reseller agreement.
- Keep the user experience helpful, not pushy.

Every practical article should support four free exits:

```text
Try this tool
Official website
Pricing page
Beginner tutorial
```

If affiliate or referral links become available later, replace the URL behind the same link label. Do not change the article into a sales page.

The user should feel:

```text
I learned the task.
I know which tool to open.
I can check the price myself.
I can follow a beginner tutorial if I need more help.
```

## Revenue Line 2: ARABAI Credits

ARABAI Credits is the future unified wallet.

The user charges ARABAI Credits and uses common AI functions through ARABAI.

Suggested top-up options:

- $10 starter
- $20 regular
- $50 creative

These are examples for product design, not final pricing.

Possible functions:

- ask a question
- write a message
- summarize a document
- generate an image
- create a 9-grid storyboard
- make a short video
- create music
- build a simple slide draft

Behind the scenes, ARABAI can connect to:

- official APIs
- AI gateway platforms
- OpenRouter-like platforms
- SiliconFlow-like platforms
- selected image, video, and audio APIs

User-facing pricing should not lead with technical words.

Beginner content should still explain important words such as token, model, API, computing power, and credits, so users understand what they are paying for.

Do not say:

```text
This costs 3,000 tokens.
```

Say:

```text
Short answer: about 1 credit.
One poster image: about 8 credits.
One short video attempt: about 80 credits.
```

For ordinary users, group credit cost by feeling:

| Task type | User-facing cost idea |
|---|---|
| Short chat, rewrite, translation | Low |
| Long document summary | Medium |
| Image generation | Medium to high |
| 9-grid storyboard | High if all frames are generated |
| Video generation | High |
| Music generation | Medium to high |

High-cost tasks should show a clear confirmation before running:

```text
This task may use many credits. Do you want to continue?
```

Do not use panic language. Just make the cost visible before the user spends.

Recommended wording:

```text
In the beginner guide, we explain what token, model, API, and computing power mean.
Here, we keep the wallet simple: you choose a task, see the estimated credits, and decide.
```

## Revenue Line 3: AI Task Marketplace

ARABAI does not sell tutorial membership.

All content remains free.

Later, a separate connected platform can allow users to post AI-related tasks.

Example tasks:

- make a product poster
- build a 9-grid video storyboard
- create a Gamma deck
- translate a product page into Arabic
- summarize a document
- write a marketing plan
- create social media content

Other people can accept and complete the tasks for payment.

This platform should be separate from the free ARABAI guide, but connected through soft links such as:

```text
Need someone to do this for you?
Post a task on the ARABAI task platform.
```

The paid part is the custom task service, not the published content.

## Key Difficulties

### Official Subscription Resale

Problem:

Official tools such as ChatGPT Plus, Gamma Pro, or Runway paid plans usually cannot be resold casually.

Solution:

Use official outbound links, affiliate links, or formal partnerships. Do not sell official subscriptions directly unless approved.

### API Cost Risk

Problem:

Image, video, and music generation can be expensive. Failed generations may still cost money.

Solution:

- estimate credit cost before generation
- require confirmation for high-cost tasks
- set daily and monthly limits
- add a margin above API cost
- log every generation cost
- start with a small model list

### User Confusion

Problem:

Ordinary users do not understand model names, tokens, context windows, or API usage.

Solution:

Explain by task:

```text
Write
Draw
Make slides
Make video
Make music
Summarize
Translate
```

### Trust

Problem:

Users may worry about paying a new platform.

Solution:

- keep content free
- make the recharge entry rare and gentle
- show official links clearly
- explain refund and credit rules
- start with small top-up amounts

## Recommended Roadmap

### Phase 1: Free Guide + Official Links

- finish practical articles
- add official tool links
- add affiliate/referral support later
- track outbound clicks
- keep all article content free
- keep pricing links separate from learning links

### Phase 2: ARABAI Credits Prototype

- create user accounts
- create wallet
- connect one text API
- connect one image API
- show credit estimates before running tasks

### Phase 3: Multimodal Credits

- add video
- add music
- add slides/doc generation
- add usage history
- add spending limits

### Phase 4: Task Marketplace

- separate platform
- user posts AI tasks
- workers accept tasks
- escrow or milestone payment
- ARABAI can earn platform commission
